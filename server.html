Here's the code:
$ sudo nohup bash -c "python -m SimpleHTTPServer 80" > server_log.txt 2>&1 &

And here's what it does:
  • sudo -enhanced privelages
  • nohup -ignore termination messages
  • python -m SimpleHTTPServer 80 -runs a simple python server
  • > server_log.txt 2>&1 -redirect output to a server log
  • & -put the process in the background.
